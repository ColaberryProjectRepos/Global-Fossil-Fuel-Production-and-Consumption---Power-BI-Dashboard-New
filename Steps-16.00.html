Fuel Sub KPIs
1. Click card visualizations twice, each for 'Fuel (P) YOY' and 'Fuel (C) YOY' from 'MeasuresForFuel'  to values field respectively.
2. For Fuel (P) YOY card, In the format->visual, in callout value, set font style =Arial, size =15, color= #DBA800, category label= off
3. In the format->general, set
Height=35, Width=80, Horizontal=55, Vertical=255, Title=off, Effects->background=off
4. For Fuel (C) YOY card, In the format->visual, in callout value, set font style =Arial, size =15, color= #827301, category label= off

Height=35, Width=90, Horizontal=180, Vertical=255, Title=off, Effects->background=off
