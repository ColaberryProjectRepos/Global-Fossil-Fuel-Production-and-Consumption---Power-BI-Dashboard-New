Production Vs Consumption (Waterfall Chart) 20

1. Before going to waterfall chart, Create few measures as mentioned below. Create the following 4 measures like we created other measures inside MeasuresForFuel table.
Coal(PvsC) = ABS([Coal(P)])-ABS([Coal(C)])
Gas(PvsC) = ABS([Gas(P)])-ABS([Gas(C)])
Oil(PvsC) = ABS([Oil(P)])-ABS([Oil(C)])
Fuel(PvsC) = ABS(MeasuresForFUEL[Fuel (P) YOY])- ABS(MeasuresForFUEL[Fuel (C) YOY])
2.  Next, create a measure description table.
3.  On the Power BI Desktop Home ribbon, Click on Enter data, name the first column header as ID and the second column header as PvsC and Name the table as Prod vs Cons.


4. Create a Measure called Variance of PvsC% to display the difference of Production and Consumption on the Waterfall chart

Now, Create Waterfall Chart using the below steps
1. Click on an empty space on the canvas, and from the visualizations pane select Waterfall Chart.
2. Drag Prod vs Cons to the category and Variance of PvsC% on Column y-axis.
3. Drag all the Tooltips = Fuel(PvsC), Coal(PvsC), Gas(PvsC), Oil(PvsC), FuelProd, FuelCons, Fuel(P)YOY, FUEL(C)YOY, CoalProd ,CoalCons, Coal(P), Coal(C), GasProd, GasCons, Gas(P), Gas(C), OilProd, OilCons, Oil(P), Oil(C),
4. In the Formatting Tab->
X-axis: On, Font =Calibri, size = 12, Color = #000000,Title = Off
Y-axis: On, Font = Calibri, size = 12, Color = #000000. Title: Off.
