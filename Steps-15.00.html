Fuel KPI
1. From visualizations click Card visualization twice, drag 'FuelProd' for first card and 'FuelCons' for second card from 'MeasuresForFuel' to values fields.
2. For both the KPI cards, In the format->visual -> callout value, set font Style = Arial, size=20, color= #F19840, display units =Auto, category label= off
3. For 1st KPI card(Fuel Prod) , In the format->general, set
Height=113, Width=136, Horizontal=0, Vertical=175, Title=Off
Effects-background=on, color=#E6E6E6, transparency=16%
4.  For 2nd KPI card (Fuel Cons) , In the format->general, set
Height=113, Width=136, Horizontal=136, Vertical=174, Title=Off
Effects-background=on, color=#E6E6E6, transparency=16%
