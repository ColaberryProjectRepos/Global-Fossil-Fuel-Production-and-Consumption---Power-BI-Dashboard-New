Year Slicer 14

1. From the visualizations pane, click on the Slicer and drag 'year' column from calendar table to values field of slicer.
2. In the format->general section, set
Outline Color=#605E5C
Outline Weight=1
Responsive=On
Height=115
Width=271
X Position = 1009
Y Position =0
Title=off
effects-background=on, transparency=0%
3.  In the format->
Set Slider to On with Color=#1A1A1A.
Set Title to On, Title Text "Year", Font Color = Black Background Color= White, Alignment=Center, Text Size=16, Bold =On, Font family=Calibri
Set Background to On, with Color =#FFFFFF and Transparency = 0%.
