KPI Indicators 17

Fuel Indicators
1. Click on the card visualizations twice, drag 'FUELPROD INDICATOR' for first card and 'FUELCONSUMP INDICATOR' for the second card from 'MeasuresForFuel' to values fields respectively.
2. For 'FUELPROD INDICATOR' card, In the format->visual, in callout value, set font size 20, color -> click fx(conditional formatting) set format style= rules, field based on = 'FUEL (P) YOY'
value >=0 and <max then #078405(Green)
value >=min and <0 then #F82A1B (Red)
category label= off
3. In the format->general, set
Height=50, Width=40, Horizontal=90, Vertical=175, Title=off, Effects-background=off
4. For 'FUELCONSUMP INDICATOR' card, In the format->visual, in callout value, set font size 20, color -> click fx(conditional formatting) set format style= rules, field based on = 'FUEL (C) YOY'
value >=0 and <max then #F82A1B(Red)
value >=min and <0 then #078405(Green)
category label= off
3. In the format->general, set
Height=50, Width=40, Horizontal=220, Vertical=175, Title=off, Effects-background=off
