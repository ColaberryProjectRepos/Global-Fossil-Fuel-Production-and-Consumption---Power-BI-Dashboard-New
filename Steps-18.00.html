Fuel Production & Consumption by Year (Line and Clustered Column Chart - Top Left) 18

  1. From the visualizations pane select Line and Clustered Column Chart.
  2. Fill up fields 
    X- axis = Year,
    Column Y-Axis = FuelProd, FuelCons, 
    Line Y-Axis = Coal(P), Coal(C), Gas(P), Gas(C), Oil(P), Oil(C)
    Tooltips = FuelProd, FuelCons, Fuel(P)YOY, FUEL(C)YOY, CoalProd ,CoalCons, Coal(P), Coal(C), GasProd, GasCons, Gas(P), Gas(C), OilProd, OilCons, Oil(P), Oil(C).
  3. In Format-> Visual->
    X-axis: Type = Continuous, Values: Font=Calibri (12),color=black, Title - off .
    Y-axis: Values: Font=Calibri (12),color=black, Title - off, Display Units=Auto. 
    Secondary Y-axis: Values: Font=Calibri (12), color=black.
    Legend: Position=Top left, Style= Marker (Circle default), Match Line Color=On, Text: Font=Calibri (12).
    Gridlines=off.
    Columns: Color-> FuelProd= #F19840 , FuelCons= #894821 
    Lines: line style= solid, join type= round, stroke width= 3px.
    Colors: Coal(P)= #009792, Coal(C)=#005E5B, Gas(P)=#F71E6A, Gas(C)= #A6053D, Oil(P)= #F300F3, Oil(C) = #7E057E.
    Markers= Off. 
    Data Labels= Off 
    Series Labels = Off
  4. In Format -> General:
    Properties, Size, Height=300, Width=505, Position, Horizontal= 270, Vertical=115. Title=On, Text=Year, Font=Calibri(15), Text color= #FFFFFF, Background color=#983800, Horizontal alignment=Center, Effects: Background=On, Background color=#E6E6E6, Transparency=20%.
    Tooltips: On, Font =Calibri, Size = 10, Label Color = #C8C8C8, Value Color =#FFFFFF.
