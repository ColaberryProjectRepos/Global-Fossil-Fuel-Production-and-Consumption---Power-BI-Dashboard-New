Global Fuel (Map) & Legend 22

  1. Click on an empty space on the canvas, and from the visualizations pane select Map.
  2. Fill up fields: 
    Location=Country, 
    Bubble Size = FuelProd
    Tooltips = FuelProd, FuelCons, Fuel(P)YOY, FUEL(C)YOY, CoalProd ,CoalCons, Coal(P), Coal(C), GasProd, GasCons, Gas(P), Gas(C), OilProd, OilCons, Oil(P), Oil(C).
  3. For Format -> visual:
    Map settings: style= grayscale, show labels=on, controls: auto zoom= on, zoom button= off.
    Colors: fx(Conditional Formatting as shown below)
     minimum color=#F2C090, middle color=#F28110, Maximum color=#813F00

5. For Format  -> General:
    Properties, Size, Height=305, Width=334, Position, Horizontal= 945, Vertical=415.            Title=On, Text=Global Fuel, Font=Calibri (15), Text color= #FFFFFF, Background                color=#983800 , Horizontal alignment=Center
    Tooltips: On, Font =Calibri, Size = 10, Label Color = #C8C8C8, Value Color =#FFFFFF.
6. To include the Map Legend, go to insert tab, click on image and select the                         Map_Legend and click open.
7. In the format pane - > general 
   properties: Height=110, Width=150, Horizontal=1040, vertical=610, title=off.
8. To include the Map legend text indicator, go to the insert tab, add on 2 text box, click      inside the box and enter 'HIGH Consumption'   'LOW Consumption', font=Calibri(11),      color=black
9. For LOW Consumption Text Box --> In the format pane->general: 
   Properties: Height=32, width=132, horizontal=982,vertical=661, title=off, effects-        background=off.
10. For HIGH Consumption Text Box --> In the format pane->general:
   Properties: Height=32, width=200, horizontal=1080,vertical=661, title=off, effects-background=off.
